# Data sources

Every dataset feeding the map, with publisher, source URL, format, local
path, as-of date, granularity, role on the map, and known caveats.

A machine-readable copy of the same table lives at
[`data/processed/dataset-manifest.csv`](data/processed/dataset-manifest.csv)
(generated by [`scripts/03_make_manifest.py`](scripts/03_make_manifest.py)).

---

## 1. Burkina Faso administrative boundaries (region level)

- **Publisher** — Institut Géographique Burkina (IGB) via OCHA Field Information Services
- **Source** — <https://data.humdata.org/dataset/cod-ab-bfa>
- **Format** — GeoJSON (admin0 / admin1 / admin2 / admin3)
- **Local path** — `data/raw/cod/bfa_admin1.geojson` (and admin0/2/3 for completeness)
- **As-of** — 2026-04-02
- **Granularity** — Polygons, admin level 1 (17 regions)
- **Role on map** — The spatial skeleton; every other layer is summarised inside these polygons
- **Notes** — Reflects the 2025 17-region reorganisation. Each feature carries `adm1_name_old` + `adm2_pcode_old`, which we use to join legacy datasets that still use the pre-2025 13-region structure.

## 2. ACLED conflict events (Burkina Faso slice)

- **Publisher** — Armed Conflict Location & Event Data Project (ACLED)
- **Source** — <https://acleddata.com> (registered Data Export Tool)
- **Format** — XLSX (weekly aggregate); per-province monthly XLSX from HDX
- **Local paths**
  - `data/raw/acled/Africa_aggregated_data_up_to_week_of-2026-04-11.xlsx` (the user-uploaded weekly export, used for Strategic developments allocation)
  - `data/raw/acled/bfa_acled_monthly_political_violence.xlsx`
  - `data/raw/acled/bfa_acled_monthly_demonstrations.xlsx`
  - `data/raw/acled/bfa_acled_monthly_civilian_targeting.xlsx`
- **Filtered output** — `data/processed/bf-acled-events.csv` (8,304 BF rows)
- **As-of** — 2026-04-11 (weekly), 2026-04 (monthly)
- **Granularity** — Per-province × week × event type (with region centroid lat/lon)
- **Role on map** — Choropleth fill (events per 100k school-age children, last 12 months) and the headline "conflict events" stat in each region popup
- **Notes** — Per-province monthly files cover Battles, Explosions/Remote violence, Violence against civilians, Protests, Riots. Strategic developments are only published at old-admin1 level; we allocate them to NEW admin1 by school-age population share within each parent old region.

## 3. Burkina Faso subnational population statistics (5-year age bands)

- **Publisher** — UNFPA / Institut National de la Statistique et de la Démographie (INSD)
- **Source** — <https://data.humdata.org/dataset/cod-ps-bfa>
- **Format** — XLSX (multi-sheet) and CSV
- **Local paths**
  - `data/raw/unfpa/bfa_admpop_2023_5yr.xlsx` (sheet `bfa_admpop_adm2_2023` carries the admin2 5-year bands)
  - `data/raw/unfpa/bfa_admpop_adm1_2023_5yr.csv`
- **As-of** — 2025-01-10 (reference year 2023)
- **Granularity** — Per-province × 5-year age band × sex
- **Role on map** — School-age population (5-14) per region; denominator of the events-per-100k-children rate
- **Notes** — Aggregated from province → new admin1 via stable `adm2_pcode`. School-age computed as `T_05_09 + T_10_14`.

## 4. OpenStreetMap schools in Burkina Faso

- **Publisher** — OpenStreetMap contributors (queried via Overpass API)
- **Source** — <https://overpass-api.de/api/interpreter> with `area["ISO3166-1"="BF"]; (node[amenity=school](area); way[amenity=school](area); relation[amenity=school](area));`
- **Format** — Overpass JSON
- **Local path** — `data/raw/osm/bfa_osm_schools.json`
- **As-of** — 2026-04-30 (live query at request time)
- **Granularity** — Point per school (named or unnamed)
- **Role on map** — Small dark dots at each school's coordinates; school count per region in popup

## 5. IDMC Internal Displacements Updates (event-level)

- **Publisher** — Internal Displacement Monitoring Centre (IDMC)
- **Source** — <https://data.humdata.org/dataset/idmc-event-data-for-bfa>
- **Format** — CSV (one row per displacement event)
- **Local path** — `data/raw/idmc/bfa_idmc_events.csv`
- **As-of** — 2026-04-30 (events 2025-11-03 → 2026-03-15)
- **Granularity** — Event level (lat/lon, start/end dates, figure, narrative)
- **Role on map** — Black triangles sized by people displaced; popup with date range and location
- **Notes** — The raw file contains two duplicate rows for the Djibasso event (same `event_id 36914`); the prep script dedupes by `event_id`. Only events that crossed IDMC's reporting threshold are included — this under-counts smaller incidents and chronic displacement.

## Data flow at a glance

```
data/raw/{acled,cod,idmc,osm,unfpa}/  ←  third-party files (gitignored)
              │
              ▼
    scripts/01_filter_acled.py        →  data/processed/bf-acled-events.csv
    scripts/02_prepare_map_data.py    →  docs/data/{regions.geojson,
                                                   events-by-region.json,
                                                   schools.json,
                                                   displacement.json}
    scripts/03_make_manifest.py       →  data/processed/dataset-manifest.csv
              │
              ▼
              docs/                   →  served by GitHub Pages from main /docs
```
